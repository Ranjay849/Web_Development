CSS Box Model and Inline & Block Elements

1. Default Box Model

Default box model: The box's total width becomes content + padding + border. So, it will take more space than the width defined.

Explanation: Width = 100px (content) + 10px (padding) + 5px (border) + 20px (margin) = 170px (total width)

2. Box-sizing: border-box

Box-sizing: border-box ensures the padding and border are included within the 100px width, so the total width is exactly 100px.

Explanation: Width remains 100px, including padding and border.
